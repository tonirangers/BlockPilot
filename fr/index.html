Sécurité & contrôle

Vous restez propriétaire. Notre rôle : opérer. Vos retraits : par signature / co-signature.

Propriété

Le coffre (Safe) est à votre nom, visible on-chain.

Retrait

Retrait possible selon la règle convenue (signature/cosignature).

Transparence

Historique on-chain + relevé périodique.

Pourquoi 1 client = 1 Safe
  • Risque isolé : pas de mélange avec d’autres clients.
  • Vous restez owner/signer : nous ne pouvons pas retirer sans votre accord.
  • Auditabilité : tout est vérifiable (Zeal / DeBank / Etherscan).
  • Retraits simples : vous signez, ça sort (pas de “ticket support”).
Modules de sécurité (adaptables)
  • Co-signature possible (seuil ajustable).
  • Limites de montant / règles d’exécution.
  • Time-lock / délai anti-panique si besoin.
  • Guard / monitoring selon profil de risque.
